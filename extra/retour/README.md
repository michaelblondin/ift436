**Proposition**: $$\sum_{j=i}^n j = \frac{(n - i + 1)(n + i)}{2}$$.

_Preuve_. Nous avons
```math
\begin{align*}
\sum_{j=i}^n j
&= \sum_{j=1}^n j - \sum_{j=1}^{i-1} j \\
&= \frac{n(n+1)}{2} - \frac{(i-1)i}{2} \\
&= \frac{n^2 + n - i^2 + i}{2} \\
&= \frac{(n - i + 1)(n + i)}{2}. &&\square
\end{align*}
```
